# column 설명  
  
Hour - 시간  
Minute - 분  
DHI - 수평면 산란일사량(Diffuse Horizontal Irradiance (W/m2))  
DNI - 직달일사량(Direct Normal Irradiance (W/m2))  
WS - 풍속(Wind Speed (m/s))  
RH - 상대습도(Relative Humidity (%))  
T - 기온(Temperature (Degree C))  
Target - 태양광 발전량 (kW)
